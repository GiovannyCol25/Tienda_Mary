
# Tienda Mary

Repositorio del proyecto "Tienda Mary" — aplicación para gestionar productos.

Descripción corta:

La aplicación incluye una API y un frontend para crear, listar, actualizar y eliminar productos.

Stack principal:

- Next.js (App Router)
- React
- TypeScript
- Node.js
- Prisma (ORM)
- Zod (validación)

Documentación de la API: `docs/API.md`.

Notas:

- No subir archivos de entorno (`.env`) ni bases de datos locales.
- `.gitignore` ya fue actualizado con exclusiones recomendadas.
